# 🎯 AssistX - AI Powered Voice and Gesture Controlled Cursor

## 📌 Project Overview
AssistX is an AI-based system that allows users to control a computer using **hand gestures and voice commands**. It provides a touchless, efficient, and user-friendly way of interaction, improving accessibility and modern human-computer interaction.

---

## 🚀 Features
- 🖐️ Gesture-based cursor movement  
- 🖱️ Mouse operations (click, right-click, scroll)  
- 🎤 Voice command execution  
- 🔊 Volume and system control  
- 📸 Screenshot capture  
- 🔄 Mode switching (Gesture / Voice)  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- MediaPipe  
- SpeechRecognition  
- PyAutoGUI
